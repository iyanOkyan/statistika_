  1. Kuartil

    Kuartil adalah ukuran pemusatan data dalam statistik yang membagi data menjadi empat bagian yang sama besar setelah data diurutkan dari nilai terkecil ke nilai terbesar. Kuartil digunakan untuk melihat penyebaran data dan mengetahui persebaran data di sekitar nilai tengah.dalam kehidupan sehari-hari masalah kuartil dapat digunakan misalnya untuk menentukan kelulusan siswa pada suatu ulangan jika ditentukan aturan siswa yang lulus adalah 25% siswa dengan nilai tertinggi.menentukan letak kuartil untuk data berkelompok, caranya sama dengan data tunggal. nilai kuartil dirumuskan sebagai berikut:

    $$Q_{i}=L_{i} + c\left ( \frac{\frac{i}{4}n - F}{f} \right )$$

    Keterangan:
    Qi = kuartil ke-i
    Li = tepi bawah kelas kuartil ke-i
    n = banyaknya data
    F = frekuensi kumulatif sebelum kelas kuartil
    f = frekuensi kelas kuartil
    C = lebar kelas

    contoh:

    $$Q_{3} = L_{3} + c \left ( \frac{\frac{3}{4}-F}{f} \right )= 75,5 + 5\left ( \frac{30-22}{9} \right )= 75,5 + 4,44 = 79,94 $$

  2. Desil

    Desil adalah salah satu konsep dalam statistik yang digunakan untuk membagi data terurut menjadi 10 bagian yang sama besar, di mana setiap bagian mencakup 10% dari seluruh data. desil sering digunakan dalam analisis distribusi data, terutama untuk menggambarkan bagaimana data tersebar atau terdistribusi di sepanjang rentang nilai. Secara teknis, desil ke-i adalah nilai pada posisi yang membagi data terurut menjadi bagian-bagian yang proporsional, yang masing-masing berisi 10% data. desil memberikan informasi yang lebih detail tentang distribusi data dibandingkan dengan ukuran pemusat lainnya seperti median atau kuartil.Cara menentukan desil digunakan rumus yang mirip dengan rumus untuk menentukan kuartil berikut

    $$D_{i} = L_{i} + c \left ( \frac{\frac{i}{10}n-F}{f} \right ) $$

    Keterangan:
    Di = desil ke-i
    Li = tepi bawah kelas desil ke-i
    n = banyaknya data
    F = frekuensi kumulatif sebelum kelas desil
    f = frekuensi kelas desil
    C = lebar kelas

    contoh:

    $$D_{2} = L_{2} + c \left ( \frac{\frac{2}{10}-F}{f} \right )= 60,5 + 5\left ( \frac{8-6}{3} \right )= 60,5 + 3,33 = 63,83 $$

  3. Persentil

    Persentil adalah ukuran statistik yang digunakan untuk menggambarkan posisi relatif suatu nilai dalam sebuah distribusi data. Persentil membagi data ke dalam seratus bagian yang sama setelah data diurutkan. Misalnya, jika sebuah nilai berada di persentil ke-80, itu berarti 80% dari data lainnya lebih kecil dari nilai tersebut, sementara 20% sisanya lebih besar.

    $$P_{i} = L_{i} + c \left ( \frac{\frac{i}{100}n-F}{f} \right ) $$

    Keterangan:
    Pi = persentil ke-i
    Li = tepi bawah kelas Presentil ke-i
    n = banyaknya data
    F = frekuensi kumulatif sebelum kelas Presentil
    f = frekuensi kelas presentil
    C = lebar kelas

    contoh:

    $$P_{50} = L_{50} + c \left ( \frac{\frac{50}{100}-F}{f} \right )= 70,5 + 5\left ( \frac{20-15}{7} \right )= 70,5 + 3,57 = 74,07 $$
